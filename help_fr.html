Introduction

L'ardoise est un logiciel de dessin dont la surface s'agrandit automatiquement quand on se déplace.
Elle est principalement destinée aux personne ayant besoin de faire un grand nombre de brouillons, ou schéma et diagrammes explicatifs utilisant une pallette de couleurs définit.
Elle est tout aussi adaptée aux enfants pour réaliser de jolis dessins.
Cependant, comme pour tout logiciel, avant de pouvoir être efficace, il faut s'accoutumer aux raccourcis clavier. C'est l'objet principal de cette aide.

Usage avancé:

L'ardoise possède un système de palette permettant de mémoriser et rétablir très facilement un pinceau : À chaque lettre du clavier est associé une brosse qui peut être soit
-vide
-une épaisseur
-une couleurs
-une couleurs et une épaisseurs

Ainsi, lorsqu'on rétablit une brosse vide, rien ne change, quand on rétablit une brosse épaisseur, seul les épaisseurs sont rétablis et les couleurs restent enchangées, pour une brosse couleur, c'est l'inverse : seules les couleurs sont rétablies et pour la brosse couleurs et épaisseurs, les deux sont rétablis.
Chaque brosse est représentée par une lettre du clavier (de [a] à [z] ) qu'il suffit de presser pour rétablir la brosse corresponlante.
Il est possible d'enregistrer la couleur actuelle dans une brosse en faisant A-(lettre), d'enregistrer l'épaisseur actuelle en faisant S-(lettre).
Remarque : le fait de réenregirtrer la même épaisseur ou la même couleur dans la même lettre réinitialise celle-ci. (voir les raccourcis)
Enfin, il est aussi possible d'enregistrer (CA-s) et d'ouvrir (CA-o) une palette.

Raccourcis

Notation:

touches modificateur (à maintenir appuyer durant toute la combinaison) :
C-(lettre) : Controle
A-(lettre) : Alt
S-(lettre) : Shift
CS-(lettre) : Controle-Shift
AS-(lettre) : Alt-Shift
CAS-(lettre) : Ctrl-Alt-Shift

les lettres seront données en minuscule.

Raccourcis standard :

C-s sauvegarder
C-o ouvrir

(
fonctions à venir :
C-c copier
C-v coller
)

Raccourcis concernant l'utilisation "simple"

<espace> Permet de changer de mode (text/dessin)
pav1 Permet de choisir la couleur du clique gauche
pav2 Permet de choisir la couleur du clique droit

C-+ / C-- Permet d'augmenter/diminuer l'épaisseur de la brosse clique gauche
A-+ / A-- Permet d'augmenter/diminuer l'épaisseur de la brosse clique droit

Raccourcis concernant l'utilisation "avancée"

A-(lettre) Permet de sauvegarder la couleur dans la lettre correspondante
S-(lettre) Permet de sauvegarder l'éapisseur dans la lettre correspondante
SA-(lettre) Permet de sauvegarder les deux pinceaux dans la lettre correspondante.

CA-s Permet de sauvegarder la palette actuelle
CA-o Permet d'ouvrir une paslette.
