Introduction

Ardoise is a drawing program whose the drawing surface is automatically expanded when needed.
It is primarily intended for peoples who need to do a lot of drafts, or schema and explanatory diagrams using a palette of defined colors.
It is also suitable for children to make pretty pictures.
However, as with any software, before to be effective, you must become accustomed to the keyboard shortcuts. This is the main subject of this help.

Advanced Usage:

The slate has a palette system to easily store and restore a brush: Each keyboard letter is assigned to a brush that can be:

  1. empty
  2. thicknesses
  3. colors
  4. colors and thicknesses

Thus, when restoring an empty brush, nothing changes. When you restore a "thick brush", only the thickness are recovered, and the colors are unchanged. For a color brush, the opposite happen : only the colors are restored. Eventually, when you restore a "colors and thickness brush", both colors and thickness are restored.
Each brush is represented by a letter key ([a] to [z]), you just need to press one to restore the corresponding brush.
It is possible to save the current colors in a brush by A-(letter), to save the current thickness by S-(letter).
Note: re-save the same thickness or the same color in the same letter resets it. (See shortcuts)
Finally, it is also possible to save (CA-s) and open (o-CA) a pallet.

Shortcuts

Notations:

modifier keys (to maintain support for any combination):
C-(letter): Control
A-(letter): Alt
S-(letter): Shift
CS-(letter): Control-Shift
AS-(letter): Alt-Shift
CAS-(letter): Ctrl-Alt-Shift

letters will be given in lowercase.

standard shortcuts:

C-s save
C-o open

(
future functions:
C-c to copy
C-v paste
)

shortcuts for Basic use

<space> Changes the mode (text / drawing)
NUM_PAD_1 Allows you to choose the color of the left click
NUM_PAD_2 Allows you to choose the color of the right click

C-+ / C-- Allows you to increase / decrease the thickness of the left click brush
A-+ / A-- Allows you to increase / decrease the thickness of the right click brush

shortcuts for Advanced use

A-(letter) Saves the color in the corresponding letter
S-(letter) Saves the thickness in the corresponding letter
SA-(letter) Saves the two brushes in the corresponding letter.

CA-s Saves the current palette
CA-o Opens a palette.
